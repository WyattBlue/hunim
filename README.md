
# HuNim

HuNim is a static site generator written in Nim.


## Usage

Start a new site:
```
hunim newsite mysite
cd mysite
```

Start the development server:
```
hunim server
```

When you are ready to deploy your site, run:

```
hunim
```

This publishing the files to the `public` directory.
